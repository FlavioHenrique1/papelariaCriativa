# Papelaria Criativa

Sistema de gestão para papelaria desenvolvido em PHP (MVC), MySQL e JavaScript puro.

## Requisitos
- PHP 8+
- MySQL
- Apache (XAMPP)

## Instalação
1. Clone o repositório
2. Copie `config.example.php` para `config.php`
3. Configure o banco
4. Importe `sql/estrutura.sql`

## Estrutura
- controllers/
- models/
- classes/
- assets/js/core
